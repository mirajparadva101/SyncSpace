Connect your Supabase project to get started

Setup Instructions

1. Go to supabase.com and create a new project

2. Go to Project Settings → API to find your URL and Anon Key

3. Go to SQL Editor and run the setup SQL (provided in README)

4. Go to Database → Replication and enable realtime for sections, texts, files tables

Required tables: sessions, sections, texts, files
